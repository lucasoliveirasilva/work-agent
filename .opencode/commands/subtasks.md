Execute o fluxo completo de subtasks no Azure DevOps:

1. Liste minhas User Stories/Tasks ativas no **projeto configurado no ambiente** que **não possuem subtasks (filhos)**.
2. Mostre a tabela numerada e pergunte qual task devo escolher.
3. Para a task escolhida, analise o conteúdo e sugira subtasks práticas.
4. Peça minha aprovação antes de criar qualquer coisa no Azure DevOps.
5. Somente após aprovação explícita, crie as subtasks como filhas da task pai.
